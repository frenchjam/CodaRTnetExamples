Represents a data stream for receiving measurement data from RTNet Server (a UDP socket)
Class Hierarchy
DataStream
class DataStream;
File
DataStream.h
Description
To create a data stream use RTNetClient::createDataStream. Packets can then be requested using the RTNetClient methods (doSingleShotAcq, monitorAcqBuffer, startAcqContinuous, requestAcqBufferPacket). To receive requested packets on the data stream use the receivePacket method.
