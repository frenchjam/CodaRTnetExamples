RTNetClient::enumerateDevices
RTNetClient
Provide array of all attached devices.
void enumerateDevices(DeviceEnum& dev) throw(NetworkException, DeviceStatusArray);
Parameters
Parameters Description
DeviceEnum& dev Reference to a DeviceEnum object to be filled with device information.
Description
Provide array of all attached devices, identified by their device ID numbers. The DeviceEnum class will be filled with an array of ID's for the attached devices.
