RTNetClient::doAutoDiscoverServer
RTNetClient
Attempt to find Codamotion Realtime Network servers on the network.
static void doAutoDiscoverServer(AutoDiscover& discover);
Parameters
Parameters Description
AutoDiscover& discover structure to be filled with array of available servers
Description
This will use a UDP auto-discovery protocol to attempt to find all available Codamotion Realtime Network servers which are attached. The results are placed in an AutoDiscover class. If it failed to find any servers, the dwNumServers member of AutoDiscover will be zero. If this happens it may be because servers are down or not connected.
