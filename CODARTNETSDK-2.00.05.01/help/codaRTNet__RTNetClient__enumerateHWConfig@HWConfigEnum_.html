RTNetClient::enumerateHWConfig
RTNetClient
Enumerate all available hardware configurations and their corresponding names
void enumerateHWConfig(HWConfigEnum& config) throw(NetworkException, DeviceStatusArray);
Parameters
Parameters Description
HWConfigEnum& config Reference to a HWConfigEnum to be filled with configuration information.
Description
Available hardware configurations are returned in a HWConfigEnum class. Each configuration has an associated address handle and an associated name. The address handle can be used to select a specific configuration in a call to startSystem, or to query the configuration using getDeviceEnable.
