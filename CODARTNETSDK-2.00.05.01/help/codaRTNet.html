Classes
Class Description
AutoDiscover An array of available Codamotion RTNet servers as found by RTNetClient::doAutoDiscoverServer
DataStream Represents a data stream for receiving measurement data from RTNet Server (a UDP socket)
DeviceEnum An array of available system devices as found by RTNetClient::enumerateDevices.
DeviceInfo Base class for retrieving runtime information about an enabled device.
DeviceInfoAlignment Use with RTNetClient::getDeviceInfo to retrieve diagnostic information about most recent Codamotion CX1 system alignment.
DeviceInfoCodaPacketMode Use with RTNetClient::getDeviceInfo to retrieve current packet settings for CODA data stream.
DeviceInfoGS16AIOInputs Use with RTNetClient::getDeviceInfo to retrieve information about GS16AIO analog interface.
DeviceInfoUnitCoordSystem Use with RTNetClient::getDeviceInfo to retrieve current CX1 unit coordinate systems.
DeviceOptions Base class for setting runtime device parameters or performing runtime setup operation on an enabled device.
DeviceOptionsAlignment Use as argument to RTNetClient::setDeviceOptions to perform a CX1 system alignment operation
DeviceOptionsCodaMode Use as argument to RTNetClient::setDeviceOptions to set Coda mode information ( CX1 sample rates and decimation).
DeviceOptionsCodaPacketMode Use as argument to RTNetClient::setDeviceOptions to set options on which packets are transmitted from the CX1 system.
DeviceOptionsGS16AIOInputs Use as argument to RTNetClient::setDeviceOptions to set GS16AIO analog input options (sample rates and decimation).
DeviceStatusArray Provides error state information for multiple RTNet devices
HWConfigEnum Array of all available hardware configurations as returned by RTNetClient::enumerateHWConfig.
NetworkException Contains error code for network connection errors
PacketDecode Abstract base class for decoding packet information received on a data stream.
PacketDecodeADC16 Decodes 16-bit measurement data received from a data stream.
RTNetClient Provides the main programming interface for commuication with Codamotion RTNet Server.
RTNetworkPacket Holds the data from a single packet on a data stream
PacketDecode3DResultExt Decodes 3D measurement packets received on a data stream.
Version Version information structure retrieved from RTNet server
