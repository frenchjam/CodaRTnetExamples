RTNetClient::doSingleShotAcq
RTNetClient
Request that a single frame of measurements be gathered on all active devices and sent to all connected data streams.
void doSingleShotAcq() throw(NetworkException, DeviceStatusArray);
Description
This will only succeed if the system has been started (using startSystem) and prepared for acquisition (using prepareForAcq). A single frame of data will be requested and, provided there are no network errors, will arrive on all connected data stream sockets.
Exceptions
NetworkException, DeviceStatusArray
